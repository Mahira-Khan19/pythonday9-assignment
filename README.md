# OOP Bank Account System

## Description

This project demonstrates Object-Oriented Programming by building a secure Bank Account System using encapsulation, inheritance, property decorators, and custom exception handling.

## Features

- Bank Account Class
- Admin Account Class
- Deposit & Withdraw
- Balance Encapsulation
- Property Decorators
- Custom Exceptions
- Inheritance
- Type Hints

## Concepts Used

- Classes & Objects
- Inheritance
- Encapsulation
- @property
- super()
- Exception Handling
- Type Hints
